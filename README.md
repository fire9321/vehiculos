# Registro de Vehiculos - Proyecto Distribuidora

Este es el programa que arme para la tarea de la semana 3. La idea es simple: una herramienta para que en la distribuidora de alimentos puedan anotar los datos de los vehiculos que van llegando (marca, modelo, motor, etc.) y tener todo ordenado.

### Como funciona
El programa corre por consola. Lo hice asi para practicar bien el uso de los comandos javac y java, asegurandome de que el codigo sea compatible con la JVM y se pueda ejecutar en cualquier equipo sin problemas.

### Lo que inclui
* Datos del vehiculo: Pide la marca, el modelo, la cilindrada, el combustible y cuantos pasajeros caben.
* Validacion: Al final muestra un resumen de todo lo que se ingreso para revisar que los datos esten bien.
* Codigo ordenado: Comente cada parte del codigo para que se entienda que hace cada variable y el escaner.

### Historia de usuario
"Como encargado de transporte, necesito una forma rapida de registrar las caracteristicas de los camiones para saber cuales mandar a cada ruta segun su capacidad".
